## üìå Project Overview
This project implements an **Anomaly-Based Intrusion Detection System (IDS)** designed to detect **Advanced Persistent Threats (APTs)**.

Unlike traditional signature-based systems, our solution uses a **Random Forest Ensemble** to identify "Living-off-the-Land" (LotL) attacks that blend in with normal traffic. The system features a novel **Dynamic Thresholding** mechanism that adjusts detection sensitivity in real-time based on network entropy, allowing it to catch "low-and-slow" data exfiltration attempts.
